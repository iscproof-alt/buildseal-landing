How do you verify release integrity?

BuildSeal turns every CI release into signed, immutable evidence.

FAQ

What is BuildSeal?

BuildSeal creates a portable proof for every release.

Why not logs?

Logs require servers. Servers disappear. Packs stay.

Can I verify offline?

Yes. Download the pack, run the verifier, get the result.

What does BuildSeal NOT guarantee?

BuildSeal proves integrity after sealing. It does not prove the build environment was honest.

Is this open?

Yes. ISC Core defines the format. Anyone can verify.
